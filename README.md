## ‚ú® Features

### Matrix Operations
- Matrix multiplication (matrix-matrix & matrix-vector)
- Matrix transpose
- Matrix inverse
- Support for multiple matrix types:
  - Dense matrices
  - Sparse matrices
  - Symmetric matrices
  - Positive definite matrices
  - Banded matrices (lower & upper)
  - Triangular matrices

### System Resolution Methods

#### Direct Methods
- LU decomposition
- Gaussian elimination
  - With pivoting
  - Without pivoting
  - Jordan method
- Cholesky decomposition

#### Iterative Methods
- Gaussian elimination
- Jacobi iteration
